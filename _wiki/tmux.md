# Tmux Cheatsheet

## Windows

`C-b c` Create window

`C-b &` Close current window


`C-b p` Previous window

`C-b n` Next window

`C-b 0...9` Select window by number

## Copy mode

`C-b [` Enter copy mode to copy text or view the history.

`C-s` then type the string to search for and press `Enter`.
  - Press `n` to next.
